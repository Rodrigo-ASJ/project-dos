## instalar con pnpm:

- Servor: pnpm add servor
- live-server: pnpm add live-server
- vite: pnpm add vite

## Ejecución pnpx:

- servor: pnpx servor src/ index.html 1234 --reload
- vite: pnpx vite src/ --port 1234

### página para ver paquetes:

$pnpm pkg set type=module

pnpm remove servor
